# LUXA Sales - Modern Sales & Inventory Management System

## About

LUXA Sales is a modern, full-featured sales and inventory management system built with Next.js, React, and Tailwind CSS. It provides a comprehensive solution for managing inventory, tracking sales, and gaining AI-powered business insights.

## Getting Started

To get started with this project, you'll need Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

## Installation

Follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm install

# Step 4: Start the development server with auto-reloading.
npm run dev
```

## Tech Stack

This project is built with:

- Next.js 15
- TypeScript
- React 18
- shadcn-ui
- Tailwind CSS
- Framer Motion
- Recharts
- React Hook Form

## Building & Deployment

To build the project for production:

```sh
npm run build
npm start
```
